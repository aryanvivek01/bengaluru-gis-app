#Create Colorized DEMs

Step 1: Navigate to your folder
cd "C:\Users\VIVEK ARYAN\Documents\bengaluru-gis-app\data\processed"

Step 2: Create a colorized DEM

We’ll use gdaldem color-relief with a color table.
First, create a text file called color.txt in the same folder with something like:

0     0 0 128
50    0 0 255
100   0 255 255
200   0 255 0
500   255 255 0
1000  255 0 0
2000  255 255 255


You can adjust the elevation breakpoints & colors (these are just an example).

Then run:

gdaldem color-relief dem_clipped.tif color.txt dem_color.tif


Now you have a colored raster.

Step 3: Create hillshade
gdaldem hillshade dem_clipped.tif dem_hillshade.tif -z 1 -s 1.0 -az 315 -alt 45

Step 4: Generate tiles (zoom 10–16)
python "C:\Users\VIVEK ARYAN\.conda\envs\blrgis\Scripts\gdal2tiles.py" -z 10-16 -r bilinear dem_color.tif dem_tiles_color
python "C:\Users\VIVEK ARYAN\.conda\envs\blrgis\Scripts\gdal2tiles.py" -z 10-16 -r bilinear dem_hillshade.tif dem_tiles_hill
